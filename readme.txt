== Description ==

Add Subtitle plugin is useful to add subtitle to pages/posts which shows in front automatically. 

No coding required.

This will work only with classic editor.

Simply download Add Subtitle plugin, activate it, and begin adding subtitle into your posts and pages today.

= Get Involved =

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can go through wordpress post/page backend.

== Frequently Asked Questions ==

 How can I use this plugin?
 You can find input option below page/post title to add subtitle.
